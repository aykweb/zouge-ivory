# にゅうめんや 象牙

架空の飲食店のサイトです。

## URL
http://ss323733.stars.ne.jp/index.php

## 制作年
2025-26年

## 使用技術
- **HTML5** / **CSS3**
- **JavaScript ES6** (Vanilla JS)
- **SQL (MySQL)** : 予約データの保存、管理
- **PHP 8.3** : 共通パーツ化、予約システムの実装
- **Figma** : デザイン制作

## 制作環境
- **MAMP**
- **phpMyAdmin**
- **macOS**
- **VS Code**
- **GitHub** : 履歴管理

## 制作のポイント
- **予約システムの構築**  
PHPとMySQLを連携し、データの送受信を実現。

- **予約の管理画面の作成**  
新規登録、一覧表示、削除操作に対応。

- **PHPによるパーツ共通化**  
ヘッダー、フッターのメンテナンス効率を向上。

- **JavaScriptによるタブ切り替えメニュー**  
メニューがスムーズに切り替わるUIを実装。
